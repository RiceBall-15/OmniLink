# OmniLink 移动端 UI 原型参考

**文档版本：** v1.0  
**更新日期：** 2026-05-13  
**设计风格：** Linear风格（暗黑极简）

---

## 🎨 设计理念

### 核心原则

1. **极简主义** - 去除一切不必要的装饰
2. **功能优先** - UI服务于功能，不喧宾夺主
3. **一致性** - 统一的视觉语言和交互模式
4. **高效性** - 减少用户操作步骤

### 视觉风格

- **色彩：** 暗黑背景 + 紫色强调色
- **字体：** Inter字体家族
- **圆角：** 统一的圆角系统（6px/8px/12px）
- **阴影：** 精致的阴影效果

## 🎯 交互设计

### 手势操作

| 手势 | 操作 | 触发区域 |
|------|------|----------|
| 左滑 | 删除会话/返回 | 会话列表项 |
| 右滑 | 返回上一页 | 聊天详情页 |
| 长按 | 显示更多操作 | 消息气泡 |
| 双击 | 快速回复 | 消息气泡 |
| 下拉 | 刷新列表 | 消息列表顶部 |
| 上滑 | 加载更多 | 消息列表底部 |

### 动画效果

1. **页面转场** - 滑动切换（300ms ease-out）
2. **消息发送** - 从底部滑入（200ms ease-out）
3. **消息接收** - 淡入 + 轻微缩放（200ms）
4. **按钮点击** - 轻微缩放反馈（100ms）
5. **列表滚动** - 惯性滚动 + 弹性效果

### 状态反馈

| 状态 | 视觉表现 | 动画 |
|------|----------|------|
| 加载中 | 骨架屏 | 闪烁 |
| 发送中 | 进度圈 | 旋转 |
| 发送成功 | 灰色勾 | 淡入 |
| 发送失败 | 红色感叹号 | 抖动 |
| 在线 | 绿色圆点 | 无 |
| 离线 | 灰色圆点 | 无 |

## 🎨 组件库

### 基础组件

#### Button 按钮

```typescript
// 主按钮
<Button variant="primary" size="large">
  登录
</Button>

// 次要按钮
<Button variant="secondary" size="medium">
  取消
</Button>

// 文字按钮
<Button variant="text" size="small">
  忘记密码？
</Button>
```

**样式规范：**
- 主按钮：紫色背景 + 白色文字
- 次要按钮：透明背景 + 边框
- 文字按钮：无背景 + 紫色文字
- 圆角：8px
- 内边距：12px 24px

#### Input 输入框

```typescript
<Input
  placeholder="请输入邮箱"
  leftIcon="mail"
  error="邮箱格式不正确"
/>
```

**样式规范：**
- 背景：`#131316`
- 边框：`#1c1c21`
- 聚焦边框：`#8b5cf6`
- 圆角：8px
- 内边距：12px 16px

#### Avatar 头像

```typescript
<Avatar size="large" source={{ uri: '...' }} />
<Avatar size="medium" name="Alice" />
<Avatar size="small" icon="person" />
```

**尺寸规范：**
- 大：64px
- 中：48px
- 小：32px
- 边框：2px solid `#1c1c21`

### 聊天组件

#### MessageBubble 消息气泡

```typescript
<MessageBubble
  content="你好，最近怎么样？"
  isOwn={false}
  time="12:30"
  status="read"
/>
```

**样式规范：**
- 收到消息：`#1c1c21`背景 + 左对齐
- 发送消息：`#8b5cf6`背景 + 右对齐
- 圆角：16px（底部4px）
- 最大宽度：80%
- 内边距：12px

#### ChatInput 聊天输入框

```typescript
<ChatInput
  onSend={handleSend}
  onImagePick={handleImagePick}
  onVoiceRecord={handleVoice}
/>
```

**功能区域：**
- 表情按钮
- 输入框
- 附件按钮
- 发送按钮

## 📐 尺寸规范

### 间距系统

| 级别 | 数值 | 用途 |
|------|------|------|
| xs | 4px | 图标与文字间距 |
| sm | 8px | 紧凑元素间距 |
| md | 12px | 默认元素间距 |
| lg | 16px | 区块间距 |
| xl | 24px | 页面边距 |
| xxl | 32px | 大区块间距 |

### 字体系统

| 级别 | 大小 | 行高 | 用途 |
|------|------|------|------|
| xs | 11px | 16px | 时间戳、标签 |
| sm | 13px | 18px | 次要文字 |
| md | 15px | 22px | 正文 |
| lg | 17px | 24px | 标题 |
| xl | 20px | 28px | 大标题 |
| xxl | 28px | 36px | 页面标题 |

### 图标尺寸

| 级别 | 尺寸 | 用途 |
|------|------|------|
| 小 | 16px | 行内图标 |
| 中 | 24px | 功能图标 |
| 大 | 32px | 空状态图标 |
