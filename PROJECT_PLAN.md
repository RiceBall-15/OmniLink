# OmniLink 项目开发计划

## 📋 项目概述

**项目名称**: OmniLink - 通用IM AI聊天应用  
**技术栈**: Rust后端 + 多平台前端  
**GitHub仓库**: git@github.com:RiceBall-15/OmniLink.git  
**开始时间**: 2026-04-29  
**目标**: 开发一个类似"微信 + OpenAI"的跨平台智能对话应用

## 🔧 开发规范

### Git工作流
- 主分支: main
- 开发分支: develop
- 功能分支: feature/xxx
- 修复分支: fix/xxx
- 每完成一个功能模块推送代码到develop分支

### 代码规范
- Rust代码遵循Rust官方规范
- 使用rustfmt格式化
- 使用clippy检查
- 每个函数必须有文档注释
- 必须编写单元测试

### 测试要求
- 核心业务逻辑覆盖率 > 80%
- 所有API必须有集成测试
- 每次提交前运行测试
